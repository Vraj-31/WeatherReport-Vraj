About This App

This website allows users to view weather information for any city by entering its name in the search bar. Key learning points from creating this project include:

  • HTML: Understanding how to structure a webpage using elements such as input fields, buttons, and sections.
  • CSS: Styling the webpage to create an attractive and responsive design, ensuring it is user-friendly across various devices.
  • JavaScript: Handling user input, fetching real-time weather data from external APIs, and dynamically displaying it without reloading the page.
  • API Integration: Learning to interact with external APIs (in this case, OpenWeather) to retrieve and present data.
  • Responsive Design: Ensuring the website adjusts its layout appropriately for different screen sizes and devices.

This website was made possible through the guidance of various YouTube channels, where I learned the fundamentals of web development step by step.
